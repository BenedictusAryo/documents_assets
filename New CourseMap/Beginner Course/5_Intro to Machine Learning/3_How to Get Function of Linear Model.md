#### **Description :**
Pada pembahasan kali ini kita akan membahas tentang fungsi dari Linier Model. 

```
y = mx + b
keterangan:
y = dependen (akibat)
x = independen (sebab)
m = slope (kemiringan garis)
b = intercept
```
